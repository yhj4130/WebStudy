■■■ JAVA Bean(자바 빈) ■■■

○ 개요
   - 자바로 작성되어진 컴포넌트(객체)들을 일반적으로 일컫는 말

○ 사용 목적
   - JSP 페이지에서 로직 부분을 분리하여 코드를 재사용함으로써
     프로그램의 구성 효율을 높이기 위함

○ JAVA Bean 만들기
   - 자바 빈은 자바 클래스이므로 자바 클래스를 설계하는 것돠 기본 규칙이 같다.
   - 자바 빈에서는 멤버 변수를 프로퍼티(Property)라고 부른다.
   - 클래스 선언은 『public』, 프로퍼티 선언은 『private』으로 한다.
   - 프로퍼티 당 하나의 getter 와 setter 를 갖는다.
   - (매개변수가 없는) 기본 생성자를 포함한다.
   - 직렬화 할 수 있어야 한다.

   ※ 반드시 인자 없는 생성자와 속성에 접근할 수 있는 get/set 메소드를 가져야 한다.
